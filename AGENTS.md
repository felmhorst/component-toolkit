# Next.js Component Toolkit

This is a Next.js component toolkit that can be used as a boilerplate for any project.

## Components

All components are located in the `components` directory, separated into the following categories:
- `animations` - animations
- `backgrounds` - background patterns or materials
- `layout` - layout components (e.g. Header, Footer, Nav-Bar)
- `ui` - generic UI components

A component folder includes:
- `index.tsx` - the component itself
- `index.scss` - the component styles
- `index.stories.tsx` - the component stories (for Storybook)
- `index.test.tsx` - the component tests
